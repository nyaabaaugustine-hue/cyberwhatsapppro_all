# Cyber WhatsApp Pro

## Bulk WhatsApp Sender Extension

Cyber WhatsApp Pro is a powerful Chrome extension for sending bulk WhatsApp messages to multiple contacts, including text, images, and videos, directly from your browser.

### Features
- Broadcast messages to multiple contacts at once
- Send text, images, and video attachments
- Schedule messages for future delivery
- Upload Excel sheets of contact numbers
- Quick reply templates
- Delivery reports and analytics
- Message customization with contact details

### Version
**1.2.9**

### Platform
Chrome / Chromium-based browsers
