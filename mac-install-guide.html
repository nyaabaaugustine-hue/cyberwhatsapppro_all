Installation steps

1
Download the extension file
Save the chrome_Extenton.zip file you received to your Mac's Downloads folder. If it came via WhatsApp, tap and hold the file → Save to Files → Downloads.
2
Unzip the file
Find the ZIP in Finder → double-click it. macOS will extract a folder called chrome_Extenton. Move this folder somewhere you won't accidentally delete it — for example your Desktop or Documents.
Keep this folder on your Mac permanently. If you move or delete it, the extension will stop working.
3
Open Chrome Extensions settings
Open Google Chrome. In the address bar, type exactly:
chrome://extensions
Then press Enter (Return ↵).
4
Turn on Developer Mode
In the top-right corner of the Extensions page, find the Developer mode toggle and switch it ON.
chrome://extensions
Developer mode
Toggle is in the top-right corner
5
Click "Load unpacked"
After turning on Developer mode, three new buttons appear at the top-left. Click Load unpacked.
Load unpacked
Pack extension
Update
6
Select the extension folder
A Finder window opens. Navigate to the chrome_Extenton folder you extracted in Step 2. Click once on the folder to select it, then click Select (or Open).
Select the folder itself — not the ZIP file and not a file inside the folder.
✓
Activate your license
Cyber WhatsApp Pro now appears in your extensions list. Click the puzzle piece 🧩 icon in Chrome's toolbar → click Cyber WhatsApp Pro → enter your activation key and click Activate Premium.
To pin the extension to your toolbar: click the puzzle piece → click the pin 📌 icon next to Cyber WhatsApp Pro.
